# 🛡️ SafeClick — Phishing Link Detection & Learning System

A machine learning-based phishing URL detection system that analyzes URLs and predicts whether they are **safe**, **suspicious**, or **phishing** links. It also educates users about phishing attacks and online safety.

## 🎯 Features

- **URL Scanner** — Paste any URL to check if it is phishing
- **Phishing Score** — Get a score from 0-100
- **Risk Level** — SAFE / SUSPICIOUS / PHISHING
- **Detailed Reasons** — Explains WHY the URL is flagged
- **Learn Section** — Tips and facts about phishing attacks
- **Machine Learning** — Trained on 70,000+ URLs
- **94-95% Accuracy** — Realistic and reliable
