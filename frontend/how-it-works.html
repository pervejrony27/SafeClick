How It Works

Understand how SafeClick analyzes URLs to detect phishing links.

🔄 Step-by-Step Process


1

You Paste a URL

Enter any suspicious URL into the scanner input box and click Scan.

2

Feature Extraction

Our system extracts 15+ features from the URL structure like length, special characters, domain info, and more.

3

ML Model Prediction

The extracted features are fed into our trained Random Forest model which was trained on 50,000+ URLs.

4

Score & Explanation

You get a phishing score (0-100) and detailed reasons explaining why the URL was flagged.

🔍 Features We Analyze

Our model looks at these URL characteristics:

📏 URL Length
🔢 Number of Dots
🌐 Has IP Address
📧 Has @ Symbol
🔒 HTTPS or HTTP
📂 URL Depth
⚠️ Suspicious Keywords
🔣 Special Characters
🏷️ Domain Length
🌍 Number of Subdomains

📊 Understanding the Score

The phishing score ranges from 0 to 100:

0 — 39
✅ SAFE
Low risk, likely legitimate
40 — 69
⚡ SUSPICIOUS
Proceed with caution
70 — 100
🚨 PHISHING
High risk, do NOT click
