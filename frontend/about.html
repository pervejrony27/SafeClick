About SafeClick

A machine learning-based phishing link detection system built as a university project.

🎣 What is Phishing?

Phishing is a type of cyber attack where attackers create fake websites that look identical to legitimate ones (like banks, social media, email providers) to steal your personal information such as passwords, credit card numbers, and other sensitive data.

🛡️ What Does SafeClick Do?

SafeClick analyzes URLs using machine learning to detect phishing links. When you paste a URL, our system extracts multiple features from the URL structure and uses a trained model to predict whether the link is safe, suspicious, or a phishing attempt. It also explains WHY the link was flagged.

💻 Technology Stack

Frontend: HTML, CSS, JavaScript
Backend: Python, Flask
Machine Learning: Scikit-learn, Random Forest
Dataset: 50,000+ phishing and legitimate URLs
Communication: REST API (JSON)
